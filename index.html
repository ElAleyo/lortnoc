Lortnoc

Welcome to the homepage of Lortnoc, a simple inversion of control framework for PHP.

Lortnoc is open source. The source code repository is hosted at GitHub.

Lortnoc is free software. It is released under the MIT license.

Lortnoc has documentation. Check out the README file to get started.

Lortnoc has unit tests. They depend on the PHPUnit framework (but the main library does not).

Lortnoc has no external dependencies. It has actually very few lines of code itself.

Lortnoc has a PEAR channel. This PEAR channel can be discovered with:

pear channel-discover scoffey.github.com/lortnoc/pear

Lortnoc supports PEAR package installation. The package may be installed with:

pear install Lortnoc
