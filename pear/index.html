Lortnoc PEAR channel

Using this Channel

This channel is to be used with the PEAR installer.

Registering the channel: 
pear channel-discover scoffey.github.com/lortnoc/pear
Listing available packages: 
pear remote-list -c lortnoc
Installing a package: 
pear install lortnoc/package_name
Installing a specific version/stability: 
pear install lortnoc/package_name-1.0.0
pear install lortnoc/package_name-beta
Receiving updates via a feed: 
http://scoffey.github.com/lortnoc/pear/feed.xml
